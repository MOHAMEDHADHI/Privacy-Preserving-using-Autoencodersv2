## Component Details

### Local Module Components

```

в”Ӯ   в”Ӯ   в””в”Җв”Җ AuxiliaryClassifier (latent вҶ’ 32 вҶ’ classes)
в”Ӯ   в””в”Җв”Җ add_differential_privacy(z, sigma)

в”Ӯ       в”ңв”Җв”Җ preprocess_data()

в”Ӯ       в””в”Җв”Җ export_latent_vectors()
в”Ӯ
в””в”Җв”Җ dataset_loader.py
    в””в”Җв”Җ load_dataset(filepath)
```
